Here we start with CSS....

Displaying the different ways how we can integrate css :

  1. Using 'style' tag

  2. Internal/Inline CSS

  3. External CSS

    4. Inline tag has higer priority than that of External css and style tag.... 

        Inline css >  style tag  >  External css

    Depeding upon which one is written first : 

         style tag  >  External css   if External css is added first and then style tag

    OR

         External css > style tag   if style tag is added first and then External css
